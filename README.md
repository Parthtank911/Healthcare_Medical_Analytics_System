# 🏥 Healthcare Medical Analytics System

## 📌 Description

This project is a Streamlit-based web application for analyzing healthcare data.
It allows users to explore patient data, visualize insights, and perform basic predictions.

---

## 🚀 Features

* 📊 Data analysis and visualization
* 📈 ECG prediction graph
* 📑 Medical report (PDF) handling
* 🤖 Basic machine learning models
* 🌐 Simple web interface using Streamlit

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib
* Scikit-learn
* Numpy
* TensorFlow
* Keras
* ReportLab
* Seaborn
* Joblib

## ▶️ How to Run

```bash id="9q9q0w"
git clone https://github.com/Parthtank911/Healthcare_Medical_Analytics_System.git
cd Healthcare_Medical_Analytics_System
pip install -r requirements.txt
streamlit run app.py
```
